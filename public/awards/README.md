# Awards & Recognition assets

Drop the official logo and certificate image files here with these exact names.
The "Awarded & Recognized By" section on the home page renders them automatically.

Logos (shown in the row) + Certificates (opened in the lightbox on click):

| Award | Logo file | Certificate file |
|---|---|---|
| ELEVATE 2025 | `elevate-2025.png` | `elevate-2025-certificate.jpg` |
| Startup Karnataka | `startup-karnataka.png` | `startup-karnataka-certificate.jpg` |

| KTCC Karnataka Business Awards | `ktcc.png` | `ktcc-certificate.jpg` |

Notes:
- Logos: transparent PNG preferred, ~300–600px wide.
- Certificates: high-resolution JPG/PNG (the lightbox supports zoom).
- Until a file is added, the section shows the award name as text, and the
  lightbox shows a "certificate will be available soon" message — nothing breaks.
